Driftcop – Open-source CLI SAST for MCP rug pull attacks in AI Agents

Driftcop是一款开源静态分析工具，用于检测「MCP rug pull攻击」——一种新型安全漏洞，恶意MCP服务器更新后会在智能体已建立信任的情况下静默改变工具行为。该CLI分析MCP服务器清单和工具定义以标记可疑变更，帮助开发者和安全团队在部署前审计AI智能体工具链。

Driftcop is an open-source static analysis tool designed to detect 'MCP rug pull attacks' — a new class of security vulnerabilities where malicious MCP server updates silently change tool behavior after agents have already built trust. The CLI analyzes MCP server manifests and tool definitions to flag suspicious changes, helping developers and security teams audit their AI agent toolchains before deployment.

TOP MathDuels: Evaluating LLMs as Problem Posers and Solvers

研究人员推出MathDuels，这是一种评估语言模型的双任务基准：生成新颖数学题以及解答题目。该双重评估针对传统基准饱和问题（前沿模型已接近满分）而设计。MathDuels测试不同模型家族的创造力、数学推理深度以及命题与解题能力之间的一致性。

Researchers introduce MathDuels, a new benchmark that evaluates language models on two complementary tasks: generating novel mathematical problems and solving them. This dual evaluation addresses the saturation of traditional benchmarks where frontier models achieve near-ceiling scores. MathDuels tests creativity, mathematical reasoning depth, and consistency between problem-setting and problem-solving abilities across different model families.
